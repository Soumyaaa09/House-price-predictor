# 🏠 House Price Predictor

A Machine Learning based web application that predicts house prices using different property features like area, bedrooms, bathrooms, location, and more.

This project combines Machine Learning, Data Analysis, and Web Development to provide accurate house price predictions through a simple and interactive user interface.

---

## 🚀 Features

- Predict house prices instantly
- User-friendly interface
- Machine Learning regression model
- Data preprocessing and analysis
- Responsive frontend design
- Flask backend integration
- Real-time prediction results

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Soumyaaa09/House-price-predictor.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd House-price-predictor
```

### 3️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

### 4️⃣ Activate Virtual Environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Mac/Linux

```bash
source .venv/bin/activate
```

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 6️⃣ Run the Application

```bash
python app.py
```

### 7️⃣ Open in Browser

```bash
http://127.0.0.1:5000
```
